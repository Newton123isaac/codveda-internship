## Task 1: Simple Calculator

### Description
A command-line calculator that performs the four basic arithmetic operations: addition, subtraction, multiplication, and division.

### Features
- Dedicated functions for each arithmetic operation
- Accepts two numeric inputs from the user
- Allows the user to select the desired operation from a menu
- Handles division by zero with a clear error message
- Validates non-numeric inputs gracefully
- Loops so multiple calculations can be done in one session

### How to Run
```bash
python simple_calculator.py
```

### Sample Output
```
========================================
       SIMPLE CALCULATOR
========================================

Operations:
  1. Addition       (+)
  2. Subtraction    (-)
  3. Multiplication (*)
  4. Division       (/)
  5. Exit

Select an operation (1-5): 1
Enter the first number : 15
Enter the second number: 7

  Result: 15.0 + 7.0 = 22
```

## Task 2: Number Guessing Game

### Description
A fun command-line game where the program randomly generates a number between 1 and 100 and the user has to guess it within a limited number of attempts.

### Features
- Uses Python's `random` module to generate a secret number
- Gives the user 7 attempts to guess correctly
- Provides "Too high" or "Too low" feedback after each guess
- Displays remaining attempts so the user stays informed
- Handles invalid inputs (letters, out-of-range numbers)
- Offers a "Play again?" option at the end of each round

### How to Run
```bash
python number_guessing.py
```

### Sample Output
```
=============================================
        NUMBER GUESSING GAME
=============================================
I'm thinking of a number between 1 and 100.
You have 7 attempts to guess it. Good luck!

Attempt 1/7 - Your guess: 50
  Too high! 📈  (6 attempt(s) remaining)

Attempt 2/7 - Your guess: 25
  Too low! 📉  (5 attempt(s) remaining)

Attempt 3/7 - Your guess: 37

🎉 Correct! You guessed it in 3 attempt(s)!
```
